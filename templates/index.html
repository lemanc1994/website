NightOut V1.3 - Updates:

1. LocalStorage Integration

New Feature: Data persistence using localStorage. Both the Tracker and Drinks Tracker pages will now save the names, scores, and added drinks. The state of the app will persist even after the page is refreshed or closed/reopened.

2. Mobile UI Enhancements

Tabs: The tabs for "Tracker", "Available People", and "Drinks Tracker" are now designed to fit on a single row on mobile, with a scroll bar if needed.

3. Available People Page:

Two people per row layout for mobile. Improved spacing between fields, preventing overlap between entries. Centered name and score fields for better alignment on mobile.

4. Tracker Page

UI Improvements: The styling for the name tags has been adjusted for a cleaner look across mobile and desktop.

5. Drinks Tracker

Drinks Buttons: The drink buttons (Larger, Wine, etc.) now match the style of the buttons on the Tracker page. LocalStorage: Added functionality to remember selected drinks even after refreshing the page.

6. CSS Updates

CSS has been adjusted for better responsiveness, including better handling of two-column layouts on mobile. Improved padding, spacing, and alignment for name tags and form fields. These changes have brought more functionality, better usability, and improved user experience across devices, especially with the new data persistence feature.
